# Gold Stock Privacy Policy

This repository contains the privacy policy for our gold price tracking mobile application.

## Privacy Policy

The full privacy policy can be found in [PRIVACY_POLICY.md](PRIVACY_POLICY.md).

## Key Points

- **No Personal Information Collected**: We do not collect, store, or process any personal information
- **Anonymous Usage Data Only**: Limited anonymous usage data is collected to improve app performance
- **App Store Compliant**: Suitable for both Android (Google Play) and iOS (Apple App Store)

## About the App

Our mobile application provides real-time gold price tracking and market information. The app is designed with privacy in mind, ensuring users can check gold prices without compromising their personal data.
